# Папка для изображений

Сюда загружайте фотографии и обложки проектов для сайта.

## Рекомендуемые имена файлов

Чтобы фото было проще подключить к `index.html`, называйте файлы так:

| Файл                | Где используется на сайте            |
|---------------------|--------------------------------------|
| `avatar.jpg`        | Фото в секции «Обо мне» (кружок)     |
| `project-mendi.jpg` | Обложка карточки «Mendi: MedTech»    |
| `project-karting.jpg` | Обложка карточки «Картинг»          |
| `project-hmi.jpg`   | Обложка карточки «HMI для крана»     |

## Как загрузить фотографии

### Через сайт GitHub (проще всего)
1. Откройте репозиторий на GitHub → папку `images`.
2. Кнопка **Add file → Upload files**.
3. Перетащите фотографии и нажмите **Commit changes**.

### Форматы
- Подойдут `.jpg`, `.png`, `.webp`.
- Обложки проектов — примерно 1200×900 px (соотношение 4:3).
- Аватар — квадрат, например 600×600 px.

После загрузки скажите мне — я подключу изображения в `index.html` вместо цветных заглушек.
